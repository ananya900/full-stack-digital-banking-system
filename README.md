# Full Stack Digital Banking System

An enterprise-grade full-stack digital banking application built using React, Spring Boot, and MySQL. The system enables secure banking operations including account management, transaction processing, and loan management with JWT-based authentication and role-based access control.

---

## Project Overview

This application is designed to simulate a real-world banking system supporting three types of users:

* Customers
* Bank Employees
* Administrators

The system ensures secure transactions, proper authorization, and structured data management.

---

## Features

### Customer

* Register and Login
* Open and Close Accounts
* View Account Details and Transaction History
* Deposit, Withdraw, and Transfer Funds
* Add and Manage Beneficiaries
* Apply for Loans
* Generate Account Statements

### Bank Employee

* Approve or Reject Account Requests
* View and Monitor Transactions
* Review and Approve Loan Applications
* Disburse Loans

### Administrator

* Manage Customer and Employee Accounts
* Activate or Deactivate Users

## How to Run

Backend:

1. Extract backend folder
2. Configure application.properties with MySQL credentials
3. Run Spring Boot application
4. Access APIs at [http://localhost:8080](http://localhost:8080)

Frontend:

1. Extract frontend folder
2. Run npm install
3. Run npm start
4. Open [http://localhost:3000](http://localhost:3000)
